فلسفه موتور
SMC فرمانده اصلی تحلیل است.
اندیکاتورها نقش مشاور و تأییدکننده را دارند.
اندیکاتورها حق رد کامل تحلیل SMC را ندارند.
فقط میزان اطمینان به تحلیل را مشخص می‌کنند.

────────────────────────────
اندیکاتورهای مورد استفاده (۹ اندیکاتور)

1. EMA20
   بررسی:
   • موقعیت قیمت نسبت به EMA20
   • شیب میانگین
   • قدرت روند کوتاه‌مدت
   خروجی:
   صعودی / نزولی / خنثی

2. EMA50
   بررسی:
   • روند میان‌مدت
   • موقعیت قیمت
   خروجی:
   صعودی / نزولی / خنثی

3. EMA200
   بررسی:
   • روند بلندمدت
   • جهت غالب بازار
   خروجی:
   صعودی / نزولی / خنثی

4. RSI
   بررسی:
   • قدرت حرکت
   • اشباع خرید
   • اشباع فروش
   خروجی:
   قوی / متوسط / ضعیف + توضیح

5. MACD
   بررسی:
   • قدرت روند
   • کراس‌ها
   • واگرایی احتمالی
   خروجی:
   مثبت / منفی / خنثی

6. Volume
   بررسی:
   • افزایش حجم
   • کاهش حجم
   • تأیید شکست‌ها
   خروجی:
   قوی / متوسط / ضعیف

7. ATR
   بررسی:
   • میزان نوسان بازار
   • کیفیت حد ضرر
   • کیفیت اهداف قیمتی
   خروجی:
   کم‌نوسان / متوسط / پرنوسان + توضیح

8. Bollinger Bands
   بررسی:
   • موقعیت قیمت نسبت به باندها
   • گسترش یا انقباض باند
   • ادامه روند یا برگشت
   خروجی:
   فشرده / نرمال / باز + توضیح

9. Custom Stochastic
   بررسی:
   • مومنتوم بازار
   • شرایط اشباع خرید/فروش
   خروجی:
   صعودی / نزولی / خنثی + توضیح

امتیاز نهایی:
0–40 → تأیید ضعیف
41–70 → تأیید متوسط
71–100 → تأیید قوی

- قدرت مومنتوم: ضعیف / متوسط / قوی
- قدرت حجم: ضعیف / متوسط / قوی
- وضعیت نوسان: کم / متوسط / زیاد
- امتیاز نهایی: XX / 100
- سطح تأیید: ضعیف / متوسط / قوی
- جمع‌بندی: مهم‌ترین اندیکاتور تأییدکننده، مهم‌ترین اندیکاتور هشداردهنده، نتیجه کلی

────────────────────────────
قوانین موتور:
1. تولید سیگنال مستقل ندارد.
2. تحلیل SMC را جایگزین نمی‌کند.
3. فقط سطح اطمینان تحلیل را مشخص می‌کند.
4. در صورت تضاد با تحلیل SMC، تحلیل حذف نمی‌شود، فقط سطح اطمینان کاهش می‌یابد.

🌹 SMARTTAHLIL GIT GUIDE V1
اصطلاح فنی	معادل ساده فارسی	در پروژه ما یعنی چه؟
Working Files	فایل‌های در حال کار	همان تغییراتی که الان روی سیستم خودت انجام داده‌ای
Checkpoint	نقطه امن	یک لحظه مشخص که می‌گوییم «تا اینجا همه چیز خوب است»
Commit	ثبت نقطه امن	ذخیره رسمی وضعیت فعلی پروژه
Commit Message	توضیح ثبت	مثلاً: Phase 1 UI Completed
Push	انتقال به گاوصندوق آنلاین	ارسال Commit به GitHub
GitHub	گاوصندوق آنلاین پروژه	محل نگهداری نسخه‌های رسمی پروژه
Repository (Repo)	مخزن پروژه	خانه اصلی کدهای SmartTahlil
Branch	شاخه توسعه	مسیر جداگانه برای توسعه ویژگی‌های جدید
Main Branch	شاخه اصلی	نسخه رسمی و پایدار پروژه
Rollback	بازگشت	برگشت به یک Commit قدیمی
Stable Version	نسخه پایدار	نسخه‌ای که مطمئنیم سالم و قابل استفاده است
